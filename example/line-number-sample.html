Line Number Sample

This file shows the Markdown input, preview, and HTML output contract used by start, line-number-skip, and line-number-set.

Basic start

Markdown

```txt {line-number-start="25"}
a
b
```

Preview

a
b

HTML

<pre><code class="language-txt" data-pre-start="25" style="counter-set:pre-line-number 25;"><span class="pre-line">a</span>
<span class="pre-line">b</span>
</code></pre>

Skip lines

Markdown

```txt {start="25" line-number-skip="2,4-5"}
a
b
c
d
e
```

Preview

a
b
c
d
e

HTML

<pre><code class="language-txt" data-pre-start="25" data-pre-line-number-skip="2,4-5" style="counter-set:pre-line-number 25;"><span class="pre-line">a</span>
<span class="pre-line pre-line-no-number">b</span>
<span class="pre-line">c</span>
<span class="pre-line pre-line-no-number">d</span>
<span class="pre-line pre-line-no-number">e</span>
</code></pre>

Set line numbers

Markdown

```txt {start="25" line-number-set="3:136,5:220"}
a
b
c
d
e
f
```

Preview

a
b
c
d
e
f

HTML

<pre><code class="language-txt" data-pre-start="25" data-pre-line-number-set="3:136,5:220" style="counter-set:pre-line-number 25;"><span class="pre-line">a</span>
<span class="pre-line">b</span>
<span class="pre-line" style="counter-set:pre-line-number 136;">c</span>
<span class="pre-line">d</span>
<span class="pre-line" style="counter-set:pre-line-number 220;">e</span>
<span class="pre-line">f</span>
</code></pre>

Skip and set

The skipped line stays blank. The next visible line can still start from the explicitly set value.

Markdown

```txt {start="1" line-number-skip="3" line-number-set="4:136"}
a
b
c
d
```

Preview

a
b
c
d

HTML

<pre><code class="language-txt" data-pre-start="1" data-pre-line-number-skip="3" data-pre-line-number-set="4:136" style="counter-set:pre-line-number 1;"><span class="pre-line">a</span>
<span class="pre-line">b</span>
<span class="pre-line pre-line-no-number">c</span>
<span class="pre-line" style="counter-set:pre-line-number 136;">d</span>
</code></pre>
